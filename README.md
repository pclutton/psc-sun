# psc-sun
This tool helps players and bookers at Paddington Sports Club (Maida Vale, London) choose the best court and time slot based on sun glare conditions. It calculates the sun's exact azimuth and elevation using the NOAA Solar Position Algorithm for any date and hour, then shows the glare risk at each baseline end for the club's two court orientations — courts 1, 2, 3, 6, 7, 8 (serving ESE ↔ WNW) and courts 4, 5, 9, 10 (serving NNE ↔ SSW). The booking view shows hourly glare dots across the full day at a glance. A separate satellite map overlay draws exact court polygons from GPS corner coordinates onto ESRI World Imagery. Designed to be pinned to an iPhone home screen via Safari's Add to Home Screen.
